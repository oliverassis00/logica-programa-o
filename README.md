# Desenvolvimento de Software 💻
<p> O conteúdo abaixo irá abordar todas as minhas anotações e conceitos estudados, no repositório, vão conter exercícios feitos na linguagem de programação Python.</p>

## Conteúdos abordados 📓

### O que é ser um Programador? 💻
<p> É a pessoa que cria instruções (código) para computadores executarem tarefas.</p>
<p> Ex: Criação de Sites, Aplicativos, Automatizações, Sistemas Empresariáis, etc...
</p>

### O que é uma IDE? 💻
<p> Ambiente Integrado de Desenvolvimento. É um programa que junta ferramentas para programar em um só lugar.
</p>
<p> Ex: Editor de Código, Executar Programas, Encontrar Erros (Debug), Sugestões Automáticas, Organização de Projetos, etc...
</p>

### O que é uma IDLE? 💻
<p> É uma ferramenta específica do Python, serve para Escrever Código Python, Executar, Testar Comandos e Aprender Programação.
</p>

### O que é Sintaxe? 💻
<p> Sintaxe é o conjunto de regras de escrita de uma Linguagem de Programação. Ela vai definir como se escreve os códigos, onde usar parênteses, onde usar chaves, identação e pontos e vírgulas.
</p>

### Algoritmos 📝
<p> Algoritmo é um conjunto finito e ordenado de instruções que resolve um problema ou executa uma tarefa.</p>
<p> Suas maneiras de representação mais comuns são Linguagem Natural, Pseudocódigo, Fluxograma e Linguagens de Programação.</p>

### Entrada, Processamento e Saída de Dados: 💻
<p> Entrada (Input) => Dados fornecidos ao sistema (teclado, mouse, sensores, apis, usuários, etc...).</p>
<p> Processamento (Process) => Ocorre a transformação dos dados conforme o algoritmo.</p>
<p> Saída (Output) => Resultado final mostrado ao usuário (Tela, impressora, print, relatório, etc..).</p>

### O que é uma Linguagem de Programação e Tipos: 🐍

#### O que é?
<p> É uma forma estruturada de escrever instruções para o computador, usando sintaxe própria, que são traduzidas para código de máquinas (0,1) e executadas.</P>

#### Tipos:

##### Compiladas:
<p> Esse tipo de linguagem transforma o código inteiro em linguagem de máquina (0, 1) antes de rodar. Código -> Compilador -> Programa.EXE -> Execução</p>
<p> Ex: C, C++, Rust, etc..
</P>

##### Interpretadas:
<p> Esse tipo de linguagem executa o código linha por linha através de um interpretador. Código -> Interpretador -> Executa</p>
<p> Ex: Pyhton, JavaScript, Ruby, etc..
</P>
